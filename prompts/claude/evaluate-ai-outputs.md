# Evaluate and Compare AI Model Outputs

**Use case:** Score and compare multiple AI-generated outputs to pick the best one or catch quality issues before publishing.
**Works with:** Claude (any version)

---

Compare these [NUMBER] AI outputs for [DESCRIBE THE TASK]:

[PASTE OUTPUT 1]
---
[PASTE OUTPUT 2]
---
[PASTE OUTPUT 3]

Evaluate each on:
- [CRITERION 1, e.g. accuracy]
- [CRITERION 2, e.g. tone match]
- [CRITERION 3, e.g. format compliance]
- [CRITERION 4, e.g. completeness]
- [CRITERION 5, e.g. brand voice adherence]

Score each 1–10 per criterion. Recommend which to use and give a one-sentence reason. Flag any hallucinations, factual errors, or rule violations.

**When to use this:**
- Running the same prompt through multiple models to find the best
- Checking AI output for hallucinations before publishing
- Evaluating whether a new model version produces better results than the current one
- QA-ing a batch of AI-generated content before it goes live
