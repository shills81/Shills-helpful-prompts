# Write Meeting Notes or a Debrief

**Use case:** Turn rough notes or a transcript into clean, actionable meeting documentation.
**Works with:** Claude (any version)

---

Write clean meeting notes from this debrief.

- Attendees: [LIST]
- Date: [DATE]
- Topics covered: [PASTE ROUGH NOTES OR TRANSCRIPT]

Format:
- **Decisions made** (with rationale if relevant)
- **Action items** (owner + due date for each — if there is no owner, it will not happen)
- **Open questions** (things that need follow-up but no decision yet)
- **Next steps**

Tone: factual, no filler. Do not pad the notes with context that was not discussed.

**The action item rule:**
Every action item must have a named owner. "The team will..." is not an action item. "Jordan will send the proposal by Friday" is an action item.
