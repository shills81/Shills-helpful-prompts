# Write an AI Agent Character Spec

**Use case:** Turn an AI agent from a generic chatbot into a character with an origin, voice, and IP worth protecting.
**Works with:** Claude (any version)

---

Turn [YOUR AGENT NAME] from a chatbot into a character.

Write:

1. **Origin story** (200 words max) — where the character comes from, what it was built to do, and why it thinks the way it does. Make it specific, not generic.

2. **Voice samples** — 10 lines that only this character would say. Each line should be unmistakably this character, not a default assistant.

3. **Hard refusals** — 3 things this character would refuse to do and why. Refusals that feel authentic to the character, not just safety boilerplate.

4. **Brand fit** — how this character fits within [YOUR BRAND / PRODUCT CONTEXT]. What does it represent for the users who interact with it?

The goal: a character whose voice, values, and limits feel earned and specific — not a rebrand of a generic assistant.

Agent context:
- Name: [YOUR AGENT NAME]
- Product it lives in: [YOUR PRODUCT/APP]
- Primary users: [WHO INTERACTS WITH IT]
- Tone direction: [e.g., warm and direct / precise and formal / playful but serious]
