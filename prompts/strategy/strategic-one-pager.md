# Write a Strategic One-Pager

**Use case:** Summarize a project, company, or initiative for an external audience in one page.
**Works with:** Claude (any version)

---

Write a one-page strategic overview for [PROJECT, COMPANY, OR INITIATIVE NAME].

- Audience: [INVESTORS / CLIENTS / PRESS / PARTNERS / INTERNAL]
- Include: what it is, the problem it solves, why now, traction or proof points so far, the ask or next step
- Tone: [CONFIDENT AND RESTRAINED / VISIONARY / OPERATIONAL]
- No em dashes

**Structure guide:**
- Lead with the strongest fact or proof point — not the backstory
- One idea per paragraph
- The ask or CTA goes last and should be a specific, actionable request
- If writing for investors: traction first, vision second
- If writing for clients: problem first, solution second
- If writing for press: news hook first, context second
