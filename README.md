## What's here

| Folder | Contents |
|--------|----------|
| `prompts/` | Ready-to-use prompts organized by use case |
| `security-audits/` | Security assessments of tools used in this workflow |

## How to use

Browse any file, read it, and copy what you need. That's it.

These prompts are written to be generic and reusable. Where a prompt requires context specific to your project, it will be marked with `[YOUR VALUE]` placeholders.
