# Contributing

This repository is a personal reference library, not a collaborative project.

## What you can do

- Read any file
- Copy any prompt into your own work (see [LICENSE](LICENSE) for terms)
- Fork the repo if you want your own private copy to modify

## What is not accepted

- Pull requests
- Feature requests
- Prompt submissions
- Issues requesting changes

Issues and pull requests opened for contributions will be closed without review.

## If you found sensitive information

Please report it via the process described in [SECURITY.md](SECURITY.md) rather than opening a public issue.
